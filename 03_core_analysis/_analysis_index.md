# 🧠 Core Analysis Index

This folder contains **supporting SQL analyses** used to answer business questions
and justify the findings presented in the executive summary.

Each file focuses on **one analytical objective**.

---

## 📊 Revenue & Growth Analysis

- **01_change_over_time.sql**  
  📈 Analyzes revenue trends over time (MoM / YoY)  
  → Supports growth consistency assessment

- **02_cumulative_analysis.sql**  
  📊 Evaluates cumulative revenue to understand growth momentum

---

## 👥 Customer Value & Retention

- **05_data_segmentation.sql**  
  🧩 Segments customers by revenue contribution (High / Mid / Low)

- **06_new_vs_returning_customers.sql**  
  🔁 Compares revenue, AOV, and order behavior of new vs returning customers

- **07_customer_retention_analysis.sql**  
  ♻️ Evaluates repeat purchase behavior and retention strength

---


## 📌 Notes
- All SQL files are **read-only analytical queries**
- Final report tables are produced separately in `/04_reporting`
- Executive conclusions are documented in `/05_insights`
